# Frontend Service

React-based UI for the e-commerce platform.

## Local Development
- Install dependencies: `pnpm install`
- Run dev server: `pnpm dev`
- Run tests: `pnpm test`
- Build: `pnpm build`

## Docker
- Build image: `docker build -t frontend-service:local .`
- Run: `docker run -p 8080:80 frontend-service:local`
